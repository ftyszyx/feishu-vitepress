微博备份工具是一个 PC 端程序。

Windows 包已经打出来了，但因为没有签名，用户下载和安装的时候，系统会提示安全风险。

<img src="/assets/LpK7bikm6oW7BDxFYcccrbPFnch.png" src-width="523" class="markdown-img" src-height="152"/>

这个提示对普通用户来说很劝退。哪怕软件本身没有问题，只要系统弹出安全警告，用户第一反应肯定是：这东西靠谱吗？

## 1.1 签名证书太贵了

我去网上查了一下 Windows 代码签名证书，常见报价一年大概要 4000 元左右。注意不是终身是每年都要交。感觉在抢钱。

后来发现一个不要钱的办法：把应用上传到 Microsoft Store，让微软的商店流程帮你完成签名和分发。

于是我决定也试一下。

## 1.2 又是一堆资料和素材

真正开始提交之后，发现这件事也没那么轻松。

需要填写各种应用资料，还要准备一堆商店素材：图标、截图、不同尺寸的展示图、说明文案等等。

好在这些图可以让 AI 帮忙生成，不然光是凑齐各种格式，就要消耗不少时间。
