# 1. 2026年1月

## 1.1 使用busuanzi统计网站点击

之前使用Umami统计网站点击，但是Umami需要自己搭建服务器，并且内存占用极高（两个服务，一个mysql,一个后台）感觉有点浪费，换成busuanzi：https://github.com/soxft/busuanzi

## 1.2 使用腾讯云的EdgeOne

之前国内的站点是直接放在阿里云上，但是因带宽和服务器资源限制，访问有点慢。

现在直接使用EdgeOne，免服务器，而且cdn加速。https://blog.bytefuse.cn/
