这事要从 2019 年说起。当时我看到一个公众号大 V 说自己在港股赚了很多钱，还推荐了一个港股券商，叫利弗莫尔证券。

这个名字本身就挺奇怪，但它有一个很吸引人的点：不需要香港银行卡，也能直接开户和转账。

我当时行动力非常强，立刻用了他的推荐码开户，然后直接汇了 1 万人民币进去。

# 1. 第一次被跨境转账教育

那也是我第一次接触这么复杂、低效的跨境转账系统。

资料要填一堆，手续费还高，整个流程慢得让人怀疑人生。钱汇出去之后，差不多一周才到账。

对于习惯了国内零延时、零手续费转账的人来说，SWIFT 这套系统真的把我恶心坏了。

不过最后总算成功上岸，钱到了港股账户。

# 2. 港股打新，一开始还挺甜

当时港股很火，大家比较流行炒新股。

港股的新股中签机制比较复杂，分散户组和机构组，中签率也会向散户倾斜一点。那几年很多创新药公司跑去港股上市，虽然还没盈利，但基本上一上市就能被炒得很高。

我也跟着参与申购新股，运气还不错，中了一只。成本大概 7000 多，暗盘的时候就翻了两倍。

如果我当时只老老实实打新股，不去做别的，收益其实还可以。

# 3. 手痒被中概股坑了

问题就出在手痒。

疫情那几年，港股里的中概股，也就是大家常说的中丐股，开始一路大跌。阿里、快手这些名字，看起来都很熟，也都像是“跌得差不多了”。

于是我开始买。

结果就是被套。一套就是好几年。

后来跌到我实在忍不住，觉得可以抄底了，但港股账户里又没钱。

# 4. 为了填坑，我跑去香港开卡

于是 2024 年，我做了一堆攻略，踏上了去香港开银行卡的路。

当时还算顺利。疫情刚过去没多久，办卡比后来容易。再加上那时候中信银行转账过去不收手续费，资金转过去也方便了很多。

于是我就陆陆续续又转了几万过去。

出于对中国互联网企业的信任，我当时觉得：这些价格再买，应该不亏吧？

结果事实证明，这些公司的股价好像没有底。

转过去的钱，基本又全被套住了。

# 5. 钱都出去了，为什么不买美股？

后来我一直在想：我是不是傻？

钱好不容易出去了，为什么不买更优秀的美股？为什么还要继续买这些让我反复受伤的中国公司？

现在回头看，港股这个市场对普通人真的不友好。除了某些阶段炒新股还有点机会，长期投资体验非常差。

我现在甚至觉得，国家不让普通人随便去炒港股，是有道理的。这个市场对普通投资者来说，就是一个坑。进去容易，出来难；看起来便宜，实际上可能越来越便宜。

# 6. 记录一下，提醒自己

写这篇，就是记录一下自己的傻蛋经历，也提醒自己：不要因为熟悉一个公司名字，就默认它值得买；不要因为跌了很多，就以为它不会继续跌；更不要因为钱已经出去了，就非得在同一个坑里继续加仓。

普通人如果没有足够的信息、研究能力和风险承受能力，真的还是少碰这些东西为妙。

港股就的垃圾，不要碰。
