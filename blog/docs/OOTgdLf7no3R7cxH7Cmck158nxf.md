# 1. 产品介绍

WeiboVault 是一个微博备份工具，可以把微博内容保存到本地，方便长期留存、离线查看和导出归档。

它既可以备份自己的微博，也可以备份其他公开或对你当前账号可见的微博内容。只要你的微博账号能正常看到，就可以尝试备份。

备份后的数据会保存在本地，你可以随时在软件里查看，也可以导出成 HTML 或 PDF 文件。

目前支持备份的内容包括：

- 微博正文
- 图片、视频等媒体资源
- 评论
- 转发内容
- 账号相关的基础信息

# 2. 使用说明

## 2.1 第一次打开：登录微博账号

第一次打开 WeiboVault 时，需要先登录自己的微博账号。软件会通过你的登录状态读取微博页面中可见的数据。

<img src="/assets/Tv9Sbm7Hyo4vvsxlxIecLKranbg.png" src-width="1272" class="markdown-img" src-height="845"/>

在弹出的窗口中完成微博登录。

<img src="/assets/Z52sbTKAaoBDM7xUA2ucjRAhnZd.png" src-width="525" class="markdown-img" src-height="747"/>

## 2.2 登录成功：进入主页

登录成功后，软件会自动打开主页。你可以在这里查看已备份账号、创建新的备份任务，也可以进入本地内容预览和导出功能。

<img src="/assets/JByzbEMAgoYb9QxSnpJcaXzRnfc.jpeg" src-width="979" class="markdown-img" src-height="647"/>

## 2.3 新建备份任务

点击「备份新账号」，可以选择要备份的微博账号类型，包括自己的微博、自己关注的账号，或者其他微博账号。

<img src="/assets/N9RTbczavoGxVHxNi5xcjW4xnce.png" src-width="257" class="markdown-img" src-height="216"/>

## 2.4 搜索并确认账号

如果要备份其他人的微博，可以通过搜索找到对应账号。

<img src="/assets/D9JXboeizorNALxWguDcT7umnQg.png" src-width="904" class="markdown-img" src-height="394"/>

点击确认后，软件会展示账号预览信息。开始备份前，建议先确认账号是否正确，避免备份错对象。

<img src="/assets/CqYfb7mQ7oodXRxcOtec4Qksn5b.png" src-width="896" class="markdown-img" src-height="717"/>

## 2.5 开始备份

确认账号后，点击「开始备份」。

<img src="/assets/RQE5bZbM8oIpfpxgX6ccdwPdnmb.png" src-width="890" class="markdown-img" src-height="152"/>

你可以根据需要选择备份配置，例如是否下载图片、视频、评论等内容。

<img src="/assets/EYPkbfcXboznm1xMmdZcq3xWnyc.png" src-width="580" class="markdown-img m-auto" src-height="301" align="center"/>

<img src="/assets/XYmebFsThorjbkxE3HXcOqiDn6e.png" src-width="866" class="markdown-img m-auto" src-height="872" align="center"/>

选好配置后，开始下载。备份过程中可以看到当前进度。

<img src="/assets/SEnqbVUFRomvo4xQSRkcC15Mnjh.png" src-width="1032" class="markdown-img" src-height="395"/>

如果临时有事，也可以暂停任务，后续再继续。

<img src="/assets/G01gbTdxOo4yMrxYc4mc9sz5nOb.png" src-width="481" class="markdown-img" src-height="232"/>

## 2.6 查看本地存档

备份完成后，可以直接在软件里预览本地内容，包括微博正文、图片、视频和评论等。

<img src="/assets/TYdpbbAxkoC4ncxkeQvcgsTWn1c.png" src-width="1233" class="markdown-img" src-height="512"/>

评论内容也会一起保存，方便后续回看。

## 2.7 导出备份

WeiboVault 支持导出 HTML 和 PDF。

更推荐使用 HTML 格式，因为 HTML 可以更好地预览图片和视频，整体查看体验也更接近网页归档。
